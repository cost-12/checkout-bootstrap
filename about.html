checkout-bootstrap

Este é um projeto de checkout desenvolvido com Bootstrap, que permite o preenchimento automático do endereço com base no CEP informado.

O projeto utiliza a API do BrasilAPI para buscar as informações de endereço a partir do CEP, proporcionando uma experiência de usuário mais rápida e eficiente.
